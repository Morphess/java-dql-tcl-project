# Java DQL and TCL Operations

This is a simple Java JDBC project that demonstrates how to work with SQL DQL and TCL operations using SQLite.

## Description

The project shows how a Java program can connect to a database, create tables, insert test data, execute SELECT queries, and manage transactions.

## Technologies

- Java
- JDBC
- SQLite
- Maven

## Features

- Create database tables
- Insert test data
- Select all records
- Select specific columns
- Use WHERE conditions
- Sort data with ORDER BY
- Limit results with LIMIT
- Join tables with JOIN
- Group data with GROUP BY
- Use aggregate functions
- Use subqueries
- Manage transactions with COMMIT, ROLLBACK, and SAVEPOINT

## Run
```text
mvn clean compile

mvn exec:java -Dexec.mainClass="Main"
```
